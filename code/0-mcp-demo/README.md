# MCP Demo

Basic MCP (Model Context Protocol) server and client examples.

## Contents

- `stdio/stdio_server.py` - MCP server with tools (weather, summarize, add, list_folders)
- `stdio/stdio_client.py` - MCP client example

## Running

```bash
# Run the server
python stdio/stdio_server.py

# Run the client (in another terminal)
python stdio/stdio_client.py
```
